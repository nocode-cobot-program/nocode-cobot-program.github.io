Towards No-Code Programming of Cobots:

Experiments with Code Synthesis by Large Code Models for Conversational Programming

Abstract

Collaborative robots” (cobots) designed to work alongside humans on assembly lines traditionally require expert programming, limiting ability to make changes, or manual guidance, limiting expressivity of the resulting programs. To address these limitations, we explore using Large Language Models (LLMs), and in particular, their abilities of doing in-context learning, for conversational code generation.
